Library Management System

Library Management System is a system which manages issuing and returning of library books and also manages the users in the library. The project aims at developing this system is to automate CRUD operations to manage well which eventually help utilizing available resources at optimum level in order to improve efficiency.It will perform the CRUD operations in library environment in easiest possible manner and presents data in a graphical and pictorial form.
